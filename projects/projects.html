Projects

CNN Invasive Species Image Classifier

Boston University

Using a binary classification approach with ReLU activation, max pooling, dropout, and a sigmoid output layer, the model was trained on large datasets of native and invasive North American plants, with the goal of automating detection via cameras, sensors, and drones for early ecosystem intrusion monitoring.

BeerMe

Boston University

Collaborators: Milo Zurn-Galinsky, Margo Miller

My team members and I created the BeerMe application which allows users to rate their favorite beers and view all other user ratings. We utilized google OAuth, mongoDB for data organization, and api calls to a large beer database.

Webex Phishing Bot with Secure XDR Integration

Cisco Systems

Collaborators: Jay Skrab, Evan Knipp, Emil Engdahl,Anya Akram

This project aimed to address human susceptibility to phishing attacks by leveraging the Webex bot API in Python to automate phishing simulations via SMTP. By tracking user interactions with phishing emails and integrating data into a conceptual XDR dashboard, administrators could analyze vulnerabilities, monitor success rates, and guide affected users toward preventative training through automated compliance messaging.
